This PR adds additional comparators for the EVM version in isoltest. These are the ones, that are now allowed:
```
// EVMVersion: >constantinople
// EVMVersion: >=constantinople
// EVMVersion: <constantinople
// EVMVersion: <=constantinople
// EVMVersion: =constantinople
// EVMVersion: !constantinople
```
