# [\#10696 Issue](https://github.com/ethereum/solidity/issues/10696) `closed`: How to write an unchecked for loop?

#### <img src="https://avatars.githubusercontent.com/u/49092?u=e839203b6d7460e1a1907d4d8071a7fe351dce67&v=4" width="50">[mudgen](https://github.com/mudgen) opened issue at [2020-12-27 23:56](https://github.com/ethereum/solidity/issues/10696):

How do I write an unchecked for loop using Solidity 0.8.0.

I tried the following but I get a syntax error:
```Solidity
uint sum;
for(uint i; i < 10; unchecked { i++; }) {
  sum += i;        
}
```

#### <img src="https://avatars.githubusercontent.com/u/504195?u=ce2facd14af9fd474ebff49f0d44891f56f7500f&v=4" width="50">[leonardoalt](https://github.com/leonardoalt) commented at [2020-12-28 09:26](https://github.com/ethereum/solidity/issues/10696#issuecomment-751647425):

```
    for(uint i; i < 10; ) {
        sum += i;        
        unchecked { i++; }
    }
````

#### <img src="https://avatars.githubusercontent.com/u/504195?u=ce2facd14af9fd474ebff49f0d44891f56f7500f&v=4" width="50">[leonardoalt](https://github.com/leonardoalt) commented at [2020-12-28 09:27](https://github.com/ethereum/solidity/issues/10696#issuecomment-751647878):

You can also wrap the whole loop:
```
    uint sum;
    unchecked {
    for(uint i; i < 10; ++i) {
        sum += i;   
    }
    }
```
