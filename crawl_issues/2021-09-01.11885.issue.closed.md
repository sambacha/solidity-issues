```solidity
contract C {
  function g() external {
    f();
  }
  
  function h() external payable {
    f();
  }

  function f() internal {
    require(msg.value == 0, "");
  }
}
```

However weird this looks, it is valid.

The SMTChecker complains with `BMC: Condition is always true.` on the `msg.value == 0` line.
